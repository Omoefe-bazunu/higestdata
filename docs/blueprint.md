# **App Name**: FinTest Platform

## Core Features:

- Buy Crypto (Mock): Simulate cryptocurrency purchase.
- Sell Crypto (Mock): Simulate cryptocurrency sale.
- Buy Gift Card (Mock): Simulate gift card purchase.
- Sell Gift Card (Mock): Simulate gift card sale.
- Airtime/Bill Payment (Mock): Simulate airtime purchase and bill payments.
- Airtime to Cash (Mock): Simulate airtime to cash conversion.
- Fraud Detection (AI Mock): Fraud Detection: Use an AI tool to monitor transaction patterns and flag suspicious activities based on mock data and pre-defined rules.

## Style Guidelines:

- Primary color: Royal blue (#1e40af) for a trustworthy and stable feel.
- Background color: Light blue (#e0eafa), a desaturated variant of the primary color, creating a calm and trustworthy atmosphere.
- Accent color: Electric green (#22c55e) for highlighting key actions and success states.
- Body font: 'Inter', a sans-serif font providing a modern, clean and objective look for the user interface.
- Headline font: 'Space Grotesk', a sans-serif font to give titles a contemporary, high-tech feel; if longer body text is anticipated, use Inter for body text
- Use a set of modern, minimalist icons to represent various financial actions and categories.
- Implement a mobile-first responsive design for accessibility across devices.
